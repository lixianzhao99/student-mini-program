# 股票分析器 V2.0 - Baostock版本使用指南

## 📌 简介

这是一个基于**多因子模型**的股票分析工具，使用**Baostock**免费数据源，可以在本地Jupyter中运行。

### 核心功能

1. **多因子评分**：动量、估值、质量、波动率等12个因子
2. **动态权重**：根据市场环境自动调整因子权重
3. **详细报告**：每只股票生成7大维度分析报告
4. **Excel导出**：一键导出分析结果

### 步骤1：安装依赖


### 步骤2：准备股票代码

股票代码格式：
- 上交所：`sh.600519`（贵州茅台）
- 深交所：`sz.000001`（平安银行）
- 创业板：`sz.300750`（宁德时代）
- 科创板：`sh.688981`（中芯国际）

### 步骤3：Jupyter中运行

#### 方式一：分析示例股票

```python

```

#### 方式二：自定义分析

```python
from stock_analyzer_baostock import custom_analysis

# 分析你想要的股票
my_stocks = [
    'sh.600519',  # 贵州茅台
    'sh.601318',  # 中国平安
    'sz.000858',  # 五粮液
    'sh.600036',  # 招商银行

custom_analysis(my_stocks)
```

#### 方式三：高级用法

```python
from stock_analyzer_baostock import StockAnalyzer

# 创建分析器
with StockAnalyzer() as analyzer:
    # 分析股票（指定日期）
    result = analyzer.analyze_stocks(
        stock_list=['sh.600519', 'sh.601318'],
        end_date='2025-01-10'
    )
    
    # 打印所有详细报告
    analyzer.print_all_reports()
    
    # 导出Excel（指定文件名）
    analyzer.export_to_excel('我的分析报告.xlsx')
```

## 📊 分析维度

### 1. 基本信息
- 当前价格
- 60日高低点
- 总市值

### 2. 估值指标
- 市盈率(PE)
- 市净率(PB)
- 估值评价

### 3. 质量指标
- 净资产收益率(ROE)
- 质量评价

### 4. 动量指标
- 短期动量(10日)
- 中期动量(20日)
- 长期动量(60日)
- 短期反转(5日)

### 5. 风险指标
- 年化波动率
- ATR指标
- Beta系数
- 历史最大回撤

### 6. 技术指标
- RSI(14日)
- 量比
- 成交量趋势

### 7. 综合评分
- 综合得分(0-1)
- 推荐等级

## ❓ 常见问题

### Q1: Baostock是什么？
A: Baostock是一个免费的A股数据接口，无需注册即可使用。

### Q2: 数据延迟吗？
A: Baostock数据有1-2天延迟，适合学习和研究使用。

### Q3: 支持哪些股票？
A: 支持所有A股（上交所、深交所、创业板、科创板）。

### Q4: 如何获取股票代码？
A: 
- 百度搜索股票名称+代码
- 使用 `bs.query_stock_basic()` 查询
