# 股票池选股器 - 详细说明文档

> 基于东方财富掘金量化平台的多因子选股策略

---

## 一、策略概述

### 1.1 核心理念

本策略是一个**多因子量化选股器**，从沪深300、中证500、上证50三大指数成分股中，通过多维度因子评分筛选出最优质的股票组成股票池。

**核心特点**：
- **纯选股**：只输出股票池和分析报告，不做回测交易
- **多因子模型**：综合动量、估值、质量、风险等多维度因子
- **智能板块选择**：根据市场环境动态选择最佳风格板块
- **详细分析报告**：每只股票生成7大维度分析报告

## 二、核心概念解释

### 2.1 什么是多因子模型？

**多因子模型**是量化投资中最常用的选股方法之一。它假设股票的收益率由多个"因子"共同决定。

**公式表示**：
```
R = α + β₁·F₁ + β₂·F₂ + ... + βₙ·Fₙ + ε
```
其中：
- R：股票预期收益率
- α：超额收益（选股能力）
- βᵢ：因子载荷（股票对因子的敏感度）
- Fᵢ：因子值（动量、估值、质量等）
- ε：特异性收益（噪声）

### 2.2 本策略使用的因子

| 因子类别 | 具体因子 | 权重 | 因子说明 |
|---------|---------|------|---------|
| **动量因子** | 短期动量(10日) | 15% | 近期价格表现 |
| | 中期动量(20日) | 20% | 中期趋势强度 |
| | 长期动量(60日) | 10% | 长期趋势确认 |
| **估值因子** | PE(市盈率) | 12% | 股价与收益比 |
| | PB(市净率) | 8% | 股价与净资产比 |
| | PS(市销率) | 5% | 股价与销售额比 |
| **质量因子** | ROE(净资产收益率) | 10% | 盈利能力 |
| | 净利润增长率 | 10% | 成长能力 |
| **风险因子** | ATR(波动率) | 10% | 波动风险控制 |

## 三、因子计算详解

### 3.1 动量因子 (Momentum)

**原理**：股票价格有惯性，过去表现好的股票未来倾向于继续表现好。

**计算方法**：
```python
动量 = (当前价格 / N日前价格) - 1
```

**本策略的三周期动量**：
- **短期动量(10日)**：`prices[-1] / prices[-11] - 1`
  - 捕捉近期强势股
  - 权重：15%

- **中期动量(20日)**：`prices[-1] / prices[-21] - 1`
  - 核心趋势判断
  - 权重：20%

- **长期动量(60日)**：`prices[-1] / prices[-61] - 1`
  - 确认长期趋势
  - 权重：10%

**注意**：使用**复权后**的价格数据，消除除权除息的影响。

### 3.2 估值因子 (Valuation)

**原理**：价格低于内在价值的股票具有投资价值。

#### 3.2.1 市盈率 (PE, Price-to-Earning Ratio)

```
PE = 每股股价 / 每股收益(EPS)
```

**解读**：
- PE越低：股价相对便宜，回本时间短
- PE越高：股价相对昂贵，预期高增长

**本策略处理**：
```python
# 分数计算：PE越低得分越高
value_pe_score = 1 - normalize(pe_ttm)
```

#### 3.2.2 市净率 (PB, Price-to-Book Ratio)

```
PB = 每股股价 / 每股净资产
```

**解读**：
- PB < 1：股价低于净资产，可能被低估
- PB > 3：市场给予较高溢价

#### 3.2.3 市销率 (PS, Price-to-Sale Ratio)

```
PS = 总市值 / 总营收
```

**适用场景**：适合营收稳定但利润波动大的成长期公司。

### 3.3 质量因子 (Quality)

**原理**：优质公司具有持续盈利能力和竞争优势。

#### 3.3.1 净资产收益率 (ROE, Return on Equity)

```
ROE = 净利润 / 净资产 = EPS / 每股净资产
```

**解读**：
- ROE > 15%：优秀，股东回报率高
- ROE > 8%：良好，具有竞争力
- ROE < 8%：一般，需要关注

#### 3.3.2 净利润增长率

```
增长率 = (本期净利润 - 上期净利润) / 上期净利润 × 100%
```

### 3.4 风险因子 (Risk/Volatility)

#### 3.4.1 ATR (Average True Range) 平均真实波动幅度

**原理**：ATR衡量价格的波动剧烈程度，帮助控制风险。

**计算步骤**：

**Step 1: 计算真实波幅 (TR)**
```
TR = MAX(最高价-最低价,
        |最高价-前一收盘价|,
        |最低价-前一收盘价|)
```

**Step 2: 计算ATR (14日平滑)**
```
ATR₁₄ = SMA(TR₁₄)  # 初始值
ATRₙ = (ATRₙ₋₁ × 13 + TRₙ) / 14  # 递推计算
```

**Step 3: 相对ATR**
```
相对ATR = ATR / 当前价格
```

**本策略应用**：
- 波动率低的股票得分高（风险低）
- 公式：`volatility_score = 1 - normalize(atr)`

### 3.5 其他技术指标

#### 3.5.1 RSI (Relative Strength Index) 相对强弱指数

**原理**：衡量价格涨跌的相对强度，判断超买超卖。

**计算公式**：
```
RSI = 100 - (100 / (1 + RS))
RS = 平均涨幅 / 平均跌幅
```

**取值范围**：0-100

**判断标准**：
- RSI > 70：超买区域，可能回调
- RSI < 30：超卖区域，可能反弹
- RSI = 50：多空平衡

#### 3.5.2 最大回撤 (Maximum Drawdown)

**原理**：历史买入后可能遭受的最大亏损。

**计算方法**：
```python
peak = prices[0]  # 历史最高点
max_dd = 0
for price in prices:
    if price > peak:
        peak = price
    dd = (price - peak) / peak  # 当前回撤
    if dd < max_dd:
        max_dd = dd  # 更新最大回撤
```

## 四、评分模型详解

### 4.1 为什么要标准化？

不同因子的量纲不同：
- PE：通常 5-100
- ROE：通常 0.01-0.50 (小数)
- 动量：-0.5 到 +0.5 (小数)

**直接相加没有意义**，需要标准化到同一尺度。

### 4.2 Min-Max 标准化

```python
def _normalize(series):
    normalized = (series - min) / (max - min)
    return normalized  # 输出范围 [0, 1]
```

**效果**：
- 最小值 → 0
- 最大值 → 1
- 中间值 → 线性插值

### 4.3 分数计算逻辑

```python
# 动量因子：越高越好
momentum_score = normalize(momentum)

# 估值因子：越低越好（PE低=便宜）
value_pe_score = 1 - normalize(pe_ttm)

# 波动率因子：越低越好（低波动=稳健）
volatility_score = 1 - normalize(atr)
```

### 4.4 综合得分计算

```python
综合得分 = (
    动量短期分 × 0.15 +
    动量中期分 × 0.20 +
    动量长期分 × 0.10 +
    估值PE分  × 0.12 +
    估值PB分  × 0.08 +
    估值PS分  × 0.05 +
    波动率分  × 0.10 +
    质量ROE分  × 0.10 +
    成长得分   × 0.10
)
```

**权重设计原则**：
- 中期动量权重最高(20%)：趋势判断的核心
- 短期动量次之(15%)：捕捉近期动量效应
- 估值因子合计25%：价值投资理念
- 风险控制10%：波动率防御

## 五、板块轮动逻辑

### 5.1 什么是风格轮动？

不同市场环境下，不同类型的股票表现不同：

| 市场环境 | 表现好的风格 | 代表指数 |
|---------|-------------|---------|
| 牛市 | 大盘蓝筹 | 上证50 |
| 震荡 | 中盘成长 | 沪深300 |
| 结构行情 | 小盘优质 | 中证500 |

### 5.2 板块选择算法

```python
def select_best_sector(end_date):
    scores = []

    for index in ['SHSE.000300', 'SHSE.000905', 'SHSE.000016']:
        # 获取指数历史数据
        data = get_index_data(index)

        # 计算各因子
        momentum = calculate_momentum(data)
        volatility = calculate_volatility(data)
        rsi = calculate_rsi(data)

        # 计算综合评分
        score = (
            momentum × 0.3 +
            (1 - volatility) × 0.3 +
            (1 - rsi/100) × 0.2 +
            volume_rank × 0.2
        )

        scores.append({'index': index, 'score': score})

    # 选择得分最高的指数
    return max(scores, key=lambda x: x['score'])['index']
```

### 5.3 市场环境判断

```python
def judge_market_environment():
    # 计算过去20日平均收益率
    avg_return = returns.mean()

    # 计算波动率
    volatility = returns.std()

    if avg_return > 0.002:
        return 'bull'      # 牛市
    elif avg_return < -0.002:
        return 'bear'       # 熊市
    elif volatility > 0.025:
        return 'volatile'   # 高波动
    else:
        return 'neutral'    # 震荡
```

## 六、选股流程详解

### 6.1 基本面筛选

```python
def filter_stocks(stock_df):
    filtered = stock_df[
        # 1. 市值门槛：排除小市值壳资源
        (stock_df['tot_mv'] >= 50) &

        # 2. PE合理性：排除亏损股和泡沫股
        (stock_df['pe_ttm'] > 0) &
        (stock_df['pe_ttm'] <= 100) &

        # 3. PB合理性：避免高估
        (stock_df['pb'] <= 10) &

        # 4. 盈利质量：ROE必须为正
        (stock_df['roe'] > 0)
    ]
    return filtered
```

**筛选目的**：
- **市值≥50亿**：排除壳资源、流动性风险
- **PE>0**：排除亏损公司
- **PE≤100**：排除泡沫
- **ROE>0**：必须盈利

### 6.2 最终选股

```python
def select_stocks(end_date):
    # 1. 选最佳板块
    best_sector = select_best_sector(end_date)

    # 2. 获取成分股
    symbols = get_constituents(best_sector)

    # 3. 获取数据
    stock_df = get_stock_data(symbols)

    # 4. 基本面筛选
    filtered = filter_stocks(stock_df)

    # 5. 多因子评分
    scored = factor_scoring(filtered)

    # 6. 取Top 20
    selected = scored.nlargest(20, '综合得分')

    return selected
```

## 十、策略局限与改进方向

### 10.1 当前局限

1. **因子维度有限**：可增加更多因子（股息率、技术因子等）
2. **静态权重**：未根据市场环境动态调整因子权重
3. **单一选股时点**：建议增加周频/日频多时点选股
4. **行业中性**：可加入行业中性化处理

### 10.2 改进建议

```python
# 1. 增加因子
factors = [
    'dividend_yield',  # 股息率
    'beta',            # 市场贝塔
    'size',            # 规模因子
    'reversal',        # 短期反转
    'volume_ratio'     # 量比
]

# 2. 动态权重调整
def adjust_weights_by_market(market_env):
    if market_env == 'bull':
        momentum_weight *= 1.3
    elif market_env == 'bear':
        quality_weight *= 1.2
    return normalized_weights

# 3. 机器学习增强
from sklearn.ensemble import RandomForestRegressor
model = RandomForestRegressor()
model.fit(X_train, y_train)
```
